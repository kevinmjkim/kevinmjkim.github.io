Inside of a Computer


In a computer, there are many different things but in this webpage there will only be main parts of the computer



RAM(Random Access Memory

  • RAM in short is anything short-term memory
  • For instance, when you run a computer, it is constantly using short-term memory to display apps and websites for you to see
  • Usually when people say "memory" they usually refer to RAM or short-term memory.



Storage

  • Storage is just what the name says it stores things such as pictures or files for a long-term period
  • Nowadays, people use solid state drives or hard drives to store photos and files
  • Storage is the opposite of RAM and it is long-term memory


CPU(Central Processing Unit)

  • The CPU is the brain of the computer and does all the computations
  • Since a lot of electricity flows through therefore it generates lots and lots of heat, it has a fan to keep it cool



Power

  • Electricity it gives energy for your computer to do stuff
  • Typically, electricity goes through the computer as alternate currents and in the computer it becomes direct current.


Motherboard

  • You can add things like RAM, SSDs, CPU, and more things on the motherboard
  • It is usually green or blue but can be red or other various colors
