101 Logic Gates


Basic Gates


AND gate

  • AND gates are when two inputs are both true or in binary, 1, the output is true
  • Even if one is false, 0, the output will be false.
  • Therefore, two inputs must be true in order for the output to be true

OR gate

  • OR gates are when one of the inputs or more are 1 then the output is true
  • Even if both are 0 then the output will be false.
  • Therefore, at least one inputs must be true in order for the output to be true

NOT gate

  • NOT gates switch the status of the input/
  • For instance, if the input is 1 then the NOT gate will make it 0 and via

XOR gate

  • XOR gates are when only one input is true for the output to be true as well, and it can't both true.
  • If both are true, then the output will be false

What's a transistor and How does it Work?

Transistor is basically an automatic switch that will turn on when electricity flows through. In a transistor, there are three components: the collector, base, and emitter

  1. Collector is collecting the electricity from the circuit
  2. The base is the input if it is provided with electricity then it is true otherwise false
  3. Emitter is just the output

AND gate creation

  1. First, you want to put the transistors right next to each other
  2. Second, you want a resistor connected to the battery and before the first transistor
  3. Third, you want a light bulb as the output after the two transistor and then another wire connected to a battery the close the circuit

OR gate creation

  1. First, you want to put one of the transistors in front of another transistor
  2. Second, a wire will be connected to each transistor separately
  3. Third, the light bulb is placed after the two transistors and a resistor is placed right before the light bulb

NOT gate creation

  1. First, you want to put the transistor with the emitter facing ground
  2. Second,you want to connect a resistor before the light bulb, which is before the transistor in the circuit as a output
  3. As a result, if the input is zero, then the light bulb turns on if the input is one, then all the electricity flows to ground

XOR gate creation

  1. First, you want to make two NOT gates with the two input connected to each gate.
  2. Second, you want the outputs of the NOT gates to be the inputs of two AND gates
  3. Third, you want a OR gate and the outputs of the two AND gates should feed into the OR gate and get the final output
  4. For instance, if they are both 1, then the light bulb will be off.
  5. However, if one is off and the another on then the light bulb will be on.
